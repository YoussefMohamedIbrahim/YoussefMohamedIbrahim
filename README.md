## 🚀 Projects

### Architecture & Machine Learning

**[KNNEngine](https://github.com/YoussefMohamedIbrahim/KNNEngine)**
High-performance KNN engine built in C++ with Python bindings via pybind11, optimized for speed and scalability.

**[LinAlg](https://github.com/YoussefMohamedIbrahim/LinAlg)**
Linear algebra library implemented from scratch in C++.

**[EigenVision](https://github.com/YoussefMohamedIbrahim/EigenVision)**
PCA + KNN digit recognition system with a Python GUI for real-time inference.

### Communications & Signal Processing

**[PAMMatchedFilter](https://github.com/YoussefMohamedIbrahim/PAMMatchedFilter)**
MATLAB/Simulink evaluation of M-PAM systems, pulse shaping, and matched filter performance in AWGN.

**[ECGOptSim](https://github.com/YoussefMohamedIbrahim/ECGOptSim)**
MATLAB pipeline comparing Bayesian optimization vs. brute-force search for ECG filtration hyperparameters.

### Hardware & Architecture

**[PipelinedARM32](https://github.com/YoussefMohamedIbrahim/PipelinedARM32)**
5-stage pipelined ARM32 core in SystemVerilog with hazard detection and forwarding.

### Databases & Systems

**Farm Management Database System**
3NF relational schema modeling inventory, finance, and operations across 15+ linked tables.
